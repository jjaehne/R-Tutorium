# R Tutorium mit Jupyter Notebook

Dieses Repository enthält interaktive Jupyter Notebooks zum Erlernen von R für Statistik und Datenanalyse.

## Inhalte

Kapitel 1 – Einführung in R  
Kapitel 2 – Datentypen und Objekte  
Kapitel 3 – Datensätze  
Kapitel 4 – Statistik  

## Voraussetzungen

Installiert:

- R
- Python
- Jupyter Notebook
- IRkernel

## Start

Terminal öffnen:

cd ~/Documents/R-Tutorium
jupyter lab

Notebook öffnen:

notebooks/Kapitel_01_Einfuehrung_R.ipynb

## Ziel

Dieses Tutorium richtet sich an Studierende der:

- Sozialwissenschaften
- Politikwissenschaft
- Statistik Anfänger
